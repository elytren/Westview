#Technologies Used 

Written with Node.js, Express, EJS, MongoDB Atlas, CSS, and Javascript 

#Problems 

I was unable to complete a shopping cart feature or view so users can't actually add anything to their carts. I would like to remedy this in the future. 

#MVP

-As a user I want the design of the overall website to be simple and look nice
-As a user I want to know what the website is about
-As a user I want to be able to see the different decades on the main page
-As a user I want there to be an easy to use navigation bar 
-As a user I want to be able to see each product, and reviews for that product
-As a user I want to be able to leave a review, edit a review and delete a review 

#Stretch Goals

-As a user I want there to be some well designed animations on the main page
-As a user I want to be able to play each decades theme song on the page
-As a user I want the cursor to look like Wanda Maximoffs powers
-As a user I want to be able to add items to my cart
-As a user I want to be able to login via Google
